# Zepto Sales EDA and Power BI Dashboard 🚀

This project performs Exploratory Data Analysis (EDA) on Zepto sales data using **SQL Server** and visualizes insights through an interactive **Power BI dashboard**.

## 📁 Project Structure

- `ZeptoSales_EDA.sql` – Clean and well-documented SQL queries used for data analysis
- `ZeptoSales_Cleaned.csv` – Cleaned version of the sales dataset with no nulls or duplicates
- `Zepto_Sales_Dashboard.pbix` – Power BI dashboard showing interactive sales insights

## ✅ Data Cleaning Summary

- No `NULL` values present in the dataset
- No duplicate rows found
- Column names were cleaned and standardized
- Price and discount values were normalized (converted from integers to decimals)

## 📊 EDA Highlights

- **Top 10 Products by MRP** identified
- **Out-of-stock vs In-stock** analysis using BIT values (1 = out of stock)
- **Total Sales Revenue by City**
- **Monthly Sales Trend**
- **Discount vs Selling Price Comparison**

## 📈 Power BI Dashboard Features

- **Interactive filters** by city, category, and time
- **KPIs** for Total Sales, Average Discount, and Number of Orders
- **Bar and Line charts** for top-performing products and trends
- **Pie chart** showing stock status distribution

## 🛠 Tools Used

- SQL Server 2019
- Power BI Desktop
- Git & GitHub
